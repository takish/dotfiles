# SEO to AIEO Refactor

既存のSEO記事をAIEO向け構造に再構成します。

## 作業内容

- 情報は削除しすぎない
- 表現は簡潔に
- 構造を最優先で再設計

## 再構成ルール

### 必須構造
1. 結論（1文）
2. 理由（3点・箇条書き）
3. 具体例 or 比較表
4. 注意点・限界
5. 向いている人 / 向いていない人

## 出力の最後に評価を付与

以下を必ず含めること：
- AIEO適合度（0〜100）
- AIに引用されやすい段落トップ3
- 改善余地がある点（最大3つ）

## 出力形式

- Markdown
- 見出しは H2 / H3
