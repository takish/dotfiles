# AIEO Optimized Content Generator

AI Overviews / ChatGPT / Gemini に引用・要約されやすい構造でコンテンツを執筆します。

## 入力情報

以下の情報を指定してください：
- テーマ
- 想定読者
- 文字数目安

## 構造ルール

### 1. 結論ファースト
- 冒頭に結論を1文で断定的に述べる
- 比喩や前置きは禁止

### 2. 理由の明示
- 結論の根拠を3点
- 箇条書きで簡潔に示す

### 3. 具体化
以下のいずれかを必ず含める：
- 比較表
- 具体例
- ケーススタディ

### 4. 注意点・限界
- 適用できないケース
- 誤解されやすいポイントを明示

### 5. 対象者の明確化
- どんな人に向いているか
- どんな人には向いていないか

## AIEO最適化ルール

- 主語を省略しない
- 曖昧語（〜と思われる、〜かもしれない）を使わない
- 見出しは断定形
- 疑問文見出しは禁止
- 箇条書き多用
- 1段落は3行以内

## 出力形式

- Markdown
- 見出しは H2 / H3 のみ
- コードブロック・表は積極的に使う
