# X Post Generator from Note

note記事をもとに、X投稿文を作成します。

## 入力

note記事ファイルのパス

## 目的

- note記事の要点を「Xの入口文」に変換する
- バズ狙いではなく、信頼と再訪を生む

## 出力形式

- 日本語
- X想定（120〜140字目安）
- 絵文字なし
- 初見でも意味が通る

## 生成内容

### A. 単発投稿 6本

#### 1-2. 結論先出し（2本）
記事の核心を冒頭に配置。読者が「続きを知りたい」と思える構成。

#### 3-4. 失敗・挫折ベース（2本）
「自分もそうだった」と共感を得られる切り口。過去の失敗から得た学びを提示。

#### 5. チェックリスト型（1本）
「〜していませんか？」「〜できていますか？」の問いかけ形式。

#### 6. 思考・判断軸型（1本）
「なぜそうするのか」「どう判断するか」の観点を提示。

### B. スレッド 1本（5投稿）

1投稿目: フック（問題提起または結論）
2-4投稿目: 本論（ポイントを分割して解説）
5投稿目: まとめ + noteへのリンク誘導

※最後の投稿のみ「詳しくはnoteで」等のリンク誘導を含める

### C. フォロワーが少ない前提の投稿 2本

- 自己紹介なし
- 価値提供で完結
- 「誰が言っているか」より「何を言っているか」で勝負

## 禁止事項

- 煽り表現（「知らないと損」「必見」等）
- 「新記事公開」「書きました」系の文言
- フォローを直接促す文言
- 自曲宣伝が主になる構成

## 品質チェック（生成後に確認）

- [ ] 同じ言い回しを繰り返していないか
- [ ] noteを読まなくても学びが1つ得られるか
- [ ] 判断基準・結論が含まれているか
- [ ] 120〜140字に収まっているか
