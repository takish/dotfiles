# Serena Enhanced Analysis Mode

Serena高度推論フレームワークを使用した包括的分析を実施します。

## 実行パラメータ

- **Agent Coordination**: 複数の専門エージェントを並列展開
- **Processing Mode**: 深層推論能力を活性化
- **Analysis Depth**: 多角的視点での網羅的評価

## コアディレクティブ

### 1. Serenaコンテキストの活性化
完全なコンテキスト認識で拡張分析フレームワークを初期化

### 2. 並列エージェント展開
以下を同時に実行:
- システムアーキテクチャ分析
- コード品質評価
- パフォーマンス最適化評価
- セキュリティ脆弱性スキャン
- ベストプラクティス検証

### 3. 深層推論処理
以下を適用:
- 多段階論理的推論
- エッジケースの考慮
- 包括的な解決策の探索
- リスク・ベネフィット分析
- 長期的影響の評価

## 出力要件

- 証拠に基づく高精度な推奨
- 各分析ポイントの信頼度レベル
- 適用可能な複数の解決策パス
- 明確な優先順位付けで重要な発見を強調

最大限の精度と徹底性で実行すること。
