# Security Check

現在の実装のセキュリティ脆弱性をチェックします。

## チェック観点

詳細なチェックリストは [references/checklist.md](references/checklist.md) を参照。

### 主要な確認項目
1. **入力検証**: SQLインジェクション、XSS、パストラバーサル
2. **認証・認可**: 権限チェック、セッション管理
3. **データ保護**: 暗号化、ログへの機密情報出力防止

## 出力形式

発見した問題は、CVSSスコアに基づく深刻度と共に報告：

```markdown
### Critical (CVSS 9.0-10.0)
[即座に対応が必要]

### High (CVSS 7.0-8.9)
[早急に対応が必要]

### Medium (CVSS 4.0-6.9)
[計画的に対応]

### Low (CVSS 0.1-3.9)
[リスクを認識した上で対応]
```

修正方法も具体的に提示すること。
