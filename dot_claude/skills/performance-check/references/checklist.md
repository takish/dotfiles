# Performance Check Checklist

## フロントエンド
- 不要な再レンダリングが発生していないか
- 重い計算処理が適切にメモ化されているか
- 画像やアセットの最適化余地
- バンドルサイズへの影響
- 遅延ローディングの活用余地
- 仮想化の必要性（長いリスト）

## バックエンド
- データベースクエリの最適化余地（N+1問題など）
- キャッシュの活用可能性
- 非同期処理の適切な使用
- メモリリークの可能性
- コネクションプールの適切な設定
- インデックスの活用

## 共通
- アルゴリズムの計算量
- 不要なループや処理
- リソースの適切な解放
- ネットワークリクエストの最適化
- 並列処理の活用可能性
