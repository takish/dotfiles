# SEO Check

現在の実装がSEOに与える影響を分析します。

## 分析観点

詳細なチェックリストは [references/checklist.md](references/checklist.md) を参照。

### 主要な確認項目
1. **テクニカルSEO**: 読み込み速度、Core Web Vitals、構造化データ
2. **コンテンツSEO**: メタタグ、見出し階層、画像alt
3. **モバイル対応**: レスポンシブ、モバイルユーザビリティ

## 出力形式

改善提案は、実装コストと期待される効果を明記：

```markdown
### 高優先度
[SEOに大きく影響・実装コスト低]

### 中優先度
[SEOに影響あり・計画的に対応]

### 低優先度
[余裕があれば対応]
```
