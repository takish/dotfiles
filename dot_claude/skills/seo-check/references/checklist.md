# SEO Check Checklist

## テクニカルSEO
- ページの読み込み速度への影響
- Core Web Vitalsへの影響（LCP, FID, CLS）
- 構造化データの実装状況
- サイトマップへの反映必要性
- robots.txtへの影響
- クロール効率

## コンテンツSEO
- メタタグの適切な設定（title, description）
- 見出しタグの階層構造（H1-H6）
- 画像のalt属性
- 内部リンク構造
- コンテンツの一意性
- キーワードの適切な配置

## モバイル対応
- レスポンシブデザインの実装
- モバイルでのユーザビリティ
- タップターゲットのサイズ
- ビューポートの設定

## その他
- canonical URLの設定
- hreflangの設定（多言語サイト）
- リダイレクトの適切性
- 404エラーの処理
- HTTPS対応
